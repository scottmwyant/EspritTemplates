Configuration Manager

On the File -> Options dialog there is a Reset button which allows you to set the current values as the default. Those values then get save to registry under HKEY_CURRENT_USER, so that each logon will have their own defaults. But what if you have several users on the same machine who want different configurations but who do not have separate logons? In Tutorial: Saving Values to the Registry it was shown how to save values to the registry. We can do the same with the values from the Configuration, and give a particular name to each set of values, so that they can be recalled as named configurations later. Not only could configurations be organized by user, but also perhaps by customers or by job types, depending upon what your requirements are.
