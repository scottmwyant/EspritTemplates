Move P0 to Any Element Center

This tutorial moves P0 to an arc or a circle center based on the users selection.

Here the very powerful and versatile GetAnyElement method of the Document is introduced. This method is used in place of GetCircle and GetArc to allow the user to pick either one. GetAnyElement returns element of type GraphicObject - is a more generic object that could be any one of the more specific objects such as an Arc or a Circle. The TypeName property is one way to determine what the data type of the object actually is. Once the type of GraphicObject has been determined, it can be set to a specific object of that type. This is referred to in programming as type casting.

C#
public void MoveP0ToAnyElementCenter()
{
Esprit.IGraphicObject element = null;
try
{
element = Document.GetAnyElement("Pick a Circle or Arc");
}
catch (System.Runtime.InteropServices.COMException)
{
// Element was not set
return;
}
if(element == null)
{
return;
}
switch (element.TypeName)
{
case "Arc":
var arc = element as Esprit.Arc;
Document.MoveP0(arc.CenterPoint);
break;
case "Circle":
var circle = element as Esprit.Circle;
Document.MoveP0(circle.CenterPoint);
break;
}
}

Select on a circle or arc when prompted and watch P0 move to that center location. Run the code again and pick on another circle or arc.
