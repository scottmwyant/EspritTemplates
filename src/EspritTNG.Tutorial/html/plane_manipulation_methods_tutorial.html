Introduction to Plane Manipulation Methods

This Tutorials shows how to modify location of an active workplane. The following code prompts the user to choose a point and then moves a workplane to that point using the Translate function.

C#
private void TranslateWorkPlaneToPointDemo()
{
Esprit.Point point = null;
try
{
point = Document.GetPoint("Select Destination Point");
}
catch (System.Runtime.InteropServices.COMException)
{
// Element was not set
return;
}
if (point == null)
{
return;
}
var plane = Document.ActivePlane;
plane.Translate(point.X - plane.X, point.Y - plane.Y, point.Z - plane.Z);
plane.Activate();
}

If you think back to the Plane properties, it has an X, Y, and Z. Instead of using the Translate method, we could have set those values directly:

C#
private void TranslateWorkPlaneToPointCoordinatesDemo()
{
Esprit.Point point = null;
try
{
point = Document.GetPoint("Select Destination Point");
}
catch (System.Runtime.InteropServices.COMException)
{
// Element was not set
return;
}
if (point == null)
{
return;
}
var plane = Document.ActivePlane;
plane.X = point.X;
plane.Y = point.Y;
plane.Z = point.Z;
plane.Activate();
}
