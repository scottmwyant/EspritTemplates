Move P0 to Circle Center

There is a Move P0 command built into ESPRIT on the Edit menu. This command prompts for a new P0 point, but you can only pick a point in response. If you want to move P0 (the XYZ origin) to the center of a Circle, for example, then you have to use SNAP mode and find the center snap location. This can sometimes be problematic, especially if the circle has a large radius. What if you just want to pick right on the Circle? This tutorial will move P0 to the Center of a chosen Circle.

Note how the GetCircle and MoveP0 methods of the Document are used. Be careful, that is MoveP0 with a zero, not the letter o after the P.

C#
public void MoveP0ToCircleCenter()
{
Esprit.Circle circle = null;
try
{
circle = Document.GetCircle("Select Reference Circle");
}
catch (System.Runtime.InteropServices.COMException)
{
// Element was not set
return;
}
if (circle != null)
{
Document.MoveP0(circle.CenterPoint);
}
}

Select on a Circle when prompted and watch the origin P0 move to the center of it.
