Scan the Graphics Collection

This GraphicsCollection is similar in concept to all of the other collections in the ESPRIT Document. This tutorial will loop through the entire GraphicsCollection and output a summary of certain information.

Notice that this code uses the Count property. Count is a value that is found on all collections. Almost always, that Count will tell you the exact number of elements in that collection. For example, Points.Count will tell you exactly how many points you have in the graphics of ESPRIT. Something to be aware of that you should notice in the results of this code is that the total number of elements that we can examine in the API is always less than the value given by the Count of the GraphicsCollection. The GraphicsCollection refers to the entire database of an ESPRIT document, and it contains many other items besides those that are exposed as graphic objects in the API. Those items are unknown to us. That is why the GraphicsCollection.Count does not reflect the total number of GraphicObject objects in the Document. We can bypass those unknown elements by testing their GraphicObjectType to see if it has a value of espUnknown.

This tutorial also introduces the use of the OutputWindow. The OutputWindow is a window in ESPRIT that can be used to display text information.

C#
public void ScanGraphicsCollection()
{
EspritApplication.EventWindow.Clear();
EspritApplication.EventWindow.Visible = true;
EspritApplication.EventWindow.AddMessage(EspritConstants.espMessageType.espMessageTypeInformation, "ScanGraphicsCollectionTutorial", $"Database contains up to {Document.GraphicsCollection.Count} GraphicObject items");
var validObjectsCount = 0;
foreach (Esprit.GraphicObject graphicObject in Document.GraphicsCollection)
{
if (graphicObject.GraphicObjectType != espGraphicObjectType.espUnknown)
{
validObjectsCount++;
EspritApplication.EventWindow.AddMessage(EspritConstants.espMessageType.espMessageTypeInformation, "ScanGraphicsCollectionTutorial", $"Element {validObjectsCount} is {graphicObject.TypeName} {graphicObject.Key}");
}
}
EspritApplication.EventWindow.AddMessage(EspritConstants.espMessageType.espMessageTypeInformation, "ScanGraphicsCollectionTutorial", $"Total Number of known GraphicObject items is {validObjectsCount}");
}
