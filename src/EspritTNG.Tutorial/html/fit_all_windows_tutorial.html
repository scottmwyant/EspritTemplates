Fit All Windows

This tutorial will loop through all of the Window objects in the Windows collection in the Document and do a fit for all of them.

C#
public void FitAllWindowsDemo()
{
WindowHelper.FitAllWindows(Document.Windows);
}
The following function is used to fit all windows:
public static void FitAllWindows(Esprit.Windows windows)
{
foreach (Esprit.Window window in windows)
{
window.Fit();
window.Refresh();
}
}

Before running this code, create some additional graphics windows in ESPRIT using Window > New Window from the menu. Arrange the Windows so that you can see several of them, such as with the Window > Tile command. Change the view in those windows and zoom in on certain areas.
