Swap Layers

Using the Layers dialog it is easy to turn all layers off or all layers on. But what if you have a lot of layers and some of them are on and some of them are off and you just need to swap them? This tutorial will loop through all of the Layer objects in the Document using a collection for Layers and turn on the ones that are off and turn off the ones that are on. Note how the Visible property of the Layer is accessed and the mathematical function Not is used to invert its Boolean value.

C#
public void SwapLayers()
{
foreach(Esprit.Layer layer in Document.Layers)
{
layer.Visible = !layer.Visible;
}
}

Before running this code, create some layers in ESPRIT or load in a file that contains several layers. Have some of those layers be on and some others off. Have the Layers dialog open so that you can see the check boxes change as this code is run.
