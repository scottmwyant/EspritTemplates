Introduction to Feature Recognition

This tutorial introduces the ESPRIT feature recognition capability to produce multiple FeatureChain and FeaturePtop objects from solid and wireframe geometry. This can be done through the API using the FeatureRecognition object.

The following code uses the CreateAutoChains method of feature recognition. When creating auto chains through the ESPRIT interface, the user defines a group and then clicks the Auto Chain icon. Suppose that you import complex DXF or IGES files, however, and they have a lot of drill holes (circles) that you do not want to put feature chains on, but you want to put features on all of the other geometry. The code shown below will group all of the arcs and segments and use the FeatureRecognition.CreateAutoChains method to create feature chains on them.

C#
public void ChainAll()
{
var set = SelectionSetHelper.GetOrAddSelectionSet(Document.SelectionSets, "Temp");
set.RemoveAll();
foreach (Esprit.IGraphicObject graphicObject in Document.GraphicsCollection)
{
if (graphicObject.GraphicObjectType == EspritConstants.espGraphicObjectType.espArc || graphicObject.GraphicObjectType == EspritConstants.espGraphicObjectType.espSegment)
{
set.Add(graphicObject);
}
}
if (set.Count > 0)
{
try
{
Document.FeatureRecognition.CreateAutoChains(set);
}
catch(System.Runtime.InteropServices.COMException)
{ }
}
Document.SelectionSets.Remove(set.Name);
}
public static SelectionSet GetOrAddSelectionSet(ISelectionSets selectionSets, string name)
{
var set = selectionSets[name] ?? selectionSets.Add(name);
return set;
}
