FeatureChain Visible Circles And Arcs

This tutorial will loop through all of the Circle objects in the Document and create features on all of the visible ones at a start angle entered by the user. It will also loop through all Arc objects in the Document and check if they are 360 degrees and visible, and if so it will create a feature on the arc at its start point. These features will be created on the same layer as the original circles and arcs.

This tutorial brings up an important topic in programming: how do you test for certain values or conditions, and how close do you have to be to satisfy that requirement? In this case we will need to test if an arc is 360 degrees. But computer languages do not think in terms of degrees, they use radians. 360 degrees is 2 * pi radians. Pi is a transcendental number with digits going out to infinity after the decimal. At some point the computer must round off 2 * pi, so our calculation will never be exact. The only way to satisfy a condition like this is to introduce some kind of tolerance, and see if what you are testing is close enough (within tolerance). There can be many approaches to this method. This tutorial demonstrates one possible solution out of many alternatives.

C#
public void FeatureChainVisibleCirclesAndArcs()
{
var angleDegree = 90.0;
if (!RequestUserInput("Enter Starting Angle", "Starting Angle", ref angleDegree))
{
return;
}
var angleRad = angleDegree * Math.PI / 180;
var previousActiveColor = Document.ActiveColor;
Document.ActiveColor = ColorHelper.ColorToUInt(Color.Blue);
//Because you cannot add a circle to a FeatureChain, make temporary arc instead
foreach (Esprit.Circle circle in Document.Circles)
{
if (circle.Layer.Visible)
{
var featureChain = Document.FeatureChains.Add(
Document.GetPoint(circle.X + circle.Radius * Math.Cos(angleRad), circle.Y + circle.Radius * Math.Sin(angleRad), circle.Z));
featureChain.Layer = circle.Layer;
featureChain.Add(Document.GetArc(circle.CenterPoint, circle.Radius, angleRad, 2 * Math.PI + angleRad));
}
}
foreach (Esprit.Arc arc in Document.Arcs)
{
if (arc.Layer.Visible)
{
var tolerance = Document.SystemUnit == espUnitType.espInch
? 0.00001
: 0.00025;
if (Math.Abs(arc.EndAngle - arc.StartAngle - 2 * Math.PI) < tolerance)
{
var featureChain = Document.FeatureChains.Add(arc.Extremity(EspritConstants.espExtremityType.espExtremityStart));
featureChain.Add(arc);
featureChain.Layer = arc.Layer;
}
}
}
Document.ActiveColor = previousActiveColor;
WindowHelper.FitAllWindows(Document.Windows);
}
The following function is used to convert the color value:
public static uint ColorToUInt(System.Drawing.Color color)
{
var value = (uint)((color.A << 24) | (color.B << 16) |
(color.G << 8) | color.R);
return value;
}

The following function is used to fit all windows:
public static void FitAllWindows(Esprit.Windows windows)
{
foreach (Esprit.Window window in windows)
{
window.Fit();
window.Refresh();
}
}

Before running this code, create some circles and arcs in ESPRIT. Create some 360-degree arcs and other arcs with are less than 360 degrees (try a 359 degree arc, for example). After running the code and you will see the newly created features in the graphics for every visible circle and 360 degree arc in the document.
